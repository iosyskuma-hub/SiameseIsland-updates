# ClaudeNotchIsland Updates

Release artifacts and Sparkle appcast for [ClaudeNotchIsland](https://github.com/iosyskuma-hub/ClaudeNotchIsland) (source repo private).

- `appcast.xml` — Sparkle feed consumed by the app
- `releases/` — signed `.zip` bundles, one per version

All items signed with EdDSA; the app verifies via embedded `SUPublicEDKey` before installing.
